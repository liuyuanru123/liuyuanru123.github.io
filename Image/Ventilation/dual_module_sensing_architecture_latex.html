Dual-module sensing architecture for ventilation volume estimation
BreathFlow — Layer 1 + Layer 2 (proposed)
(a)Module A
Window aperture sensor
ultrasonic + magnetic reed
(b)Module B
Ventilation & IEQ sensor stack
anemometer + CO₂ / PM / T / RH / MRT
(c)Signals from Module A
· Aperture area $A(t)$ [m²]
· Effective opening height $H_{\mathrm{eff}}$
· Open / closed binary state
· Opening duration $\Delta t_{\mathrm{open}}$
sampling: 1 Hz, on-device
(d)Signals from Module B
· Wind speed $v(t)$ [m/s]
· Indoor T, Outdoor T → $\Delta T$
· Globe temperature → $\overline{T}_{\mathrm{r}}$ (MRT)
· RH (in / out), CO₂ ppm
· PM2.5, PM10 (in / out)
· CO₂ decay slope (tracer)
sampling: 1 Hz, edge MCU
(e)Physics-based estimator
Buoyancy-driven (stack):
$\displaystyle Q_{b} \;=\; C_{d}\, A(t)\, \sqrt{\frac{2\,g\,\Delta T\,H}{T}}$
Wind-driven (cross):
$\displaystyle Q_{w} \;=\; C_{w}\, A(t)\, v(t)$
Combined:
$\displaystyle Q_{\mathrm{phys}} \;=\; \sqrt{\, Q_{b}^{2} + Q_{w}^{2}\,}$
calibrated by CO₂-decay ground truth
(f)LSTM fusion — residual + temporal
Input window: last 10 min · Target: ACH [h⁻¹]
Feature vector $\mathbf{x}_{t} \;=\; \bigl[\, A,\; v,\; \Delta T,\; \mathrm{RH},\; \mathrm{CO}_{2},\; \mathrm{slope},\; Q_{\mathrm{phys}} \,\bigr]$
$\displaystyle \widehat{\mathrm{ACH}} \;=\; Q_{\mathrm{phys}} \;+\; \mathrm{LSTM}_{\boldsymbol{\theta}}\!\left(\mathbf{x}_{t-N:t}\right)$
calibrated by CO₂-decay ground truth
(g)Output to BreathFlow Layer 2
Estimated ACH:$\widehat{\mathrm{ACH}}(t)\;\;[\mathrm{h}^{-1}]$ Confidence:$\sigma$  from LSTM dropout ensemble Feeds:→ PMV / adaptive comfort predictor  ·  decision layer (open / hybrid / AC)
